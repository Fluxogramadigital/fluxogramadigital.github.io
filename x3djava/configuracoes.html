Configurações

Existem alguns passos que são necessários completar de maneira a preparar o computador para a execução de aplicações do X3D integrado com Java.

 

Preparação do computador para usar SAI

Vai de seguida proceder-se à explicação dos passos necessários para preparar o computador para editar e executar aplicações usando SAI.

  1. Instalar o Xj3D installer for Windows que já possui o Java 1.4.2_06. Com esta acção vão ser instalados o browser Xj3D e o JAVA2 Runtime enviroment SE V1.4.2_07. (http://www.xj3d.org/download.html)
  1. Instalar o Netbeans.
  1. Para compilar uma aplicação Java em que se use o SAI necessitamos de usar a package org.web3d.x3d.sai. Para a podermos utilizar, necessitamos de definir a classpath do Java. O browser Xj3D possui os jars necessários para o package org.web3d.x3d.sai, pelo que basta copiar estes ficheiros do local de instalação do xj3d: “C:\Programas\Xj3D\jars” (por defeito), para a directoria: C:\j2sdk1.4.2_04\jre\lib\ext (por defeito), criada na instalação do Netbeans. Ao executar o Netbeans os ficheiros vão ser acedidos automaticamente, permitindo a utilização da package org.web3d.x3d.sai. Este passo não necessita de ser realizado se não pretendermos compilar nenhuma aplicação.
  1. Para usar aplicações Java com X3D através de SAI, usando o browser xj3d necessitamos de ter estas aplicações na directoria de instalação do Xj3D. Ao instalar o Xj3D é criada uma directoria “examples” (C:\Programas\Xj3D\examples). As aplicações Java devem ser colocadas dentro de uma pasta, nesta directoria.
  1. Para executar a aplicação Java é usado um ficheiro bat. No caso especifico da aplicação desenvolvida, este ficheiro vai atribuir mais memória ao Java Virtual Machine, especificar as bibliotecas necessárias e executa a aplicação Java. Para cada aplicação Java diferente deverá editar-se este ficheiro bat e modificar o nome da aplicação Java para a aplicação actual.

 

Preparação o computador para usar nós de som

Para executar som numa cena X3D é necessário a instalação do OpenAL, biblioteca muito utilizada em aplicações e jogos que necessitem de áudio 3D.

Deve fazer-se o download do OpenAL11CoreSDK.exe da página do OpenAL e instalar este software. (http://www.openal.org/)
