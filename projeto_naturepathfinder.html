NaturePathfinder

This project is an electronic Road-Book for smartphones, consisting of two complementary applications.

The first one is a "Graphical Editor" application that allows the user to draw up the notes of a Road-Book directly on the ground, collecting data such as total and partial mileage, directly from the smartphone's GPS. The second one is an application that assists the user in navigating with an existing Road-Book in a database.

The main goal of the application is to automatically display the waypoints by detecting their proximity and notifying the user if they are making a navigation error.
